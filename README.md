## Getting Started

### 1. Install gulp globally:

```
npm install -g gulp
```

### 2. Install gulp and gulp-util in your project devDependencies:

```
npm install
```


### 3. Run gulp

```
gulp
```

The default tasks will run and gulp will watch for changes in stylus files.
